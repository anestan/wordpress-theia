== Description ==
A widget that allows for quick and easy testing of multiple woocommerce widgets. Suggested for woocommmerce developer

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the woocommerce-monster-widget folder to the to the /wp-content/plugins/ directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. You can see Woocommerce Monster Widget at the widget page
4. Drag and Drop theme to your widget spot
5. This is need Woocommerce plugin to make it run.
